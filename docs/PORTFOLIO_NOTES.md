# Portfolio Notes

This repository complements the `bangladesh-climate-wage` repository by showing a different part of the research skill set:

- remote sensing
- Google Earth Engine
- geospatial data engineering
- economic damage accounting
- hydroclimatic driver analysis
- reproducible Python workflows

## Suggested GitHub Description

Satellite-based riverbank erosion and economic damage-accounting workflow.

## Suggested Topics

`remote-sensing`, `google-earth-engine`, `riverbank-erosion`, `geospatial`, `damage-accounting`, `climate-risk`, `python`, `landsat`, `sentinel-2`, `reproducible-research`

## What Reviewers Should Notice

- The code is organized by export, processing, and analysis stages.
- Raw data and generated outputs are intentionally excluded.
- The method is reusable across river systems, even though the current documentation includes a Bangladesh case study.
- The repository shows version-control discipline through documentation, `.gitignore`, and staged public outputs.
