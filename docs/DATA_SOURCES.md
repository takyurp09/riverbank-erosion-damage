# Data Sources

This file lists the main data families used by the workflow. It is a guide for adaptation, not a claim that all sources are required for every case study.

## Remote Sensing

- Landsat 5/7/8/9 surface reflectance
- Sentinel-2 Level-2A imagery
- JRC Global Surface Water YearlyHistory
- VIIRS and DMSP nighttime lights

## Population and Built Environment

- WorldPop population grids
- GHSL population and built-up layers
- Google Open Buildings
- Microsoft Global Building Footprints
- OpenStreetMap roads, embankments, facilities, and infrastructure

## Agricultural and Ecosystem Exposure

- SPAM crop production and harvested-area grids
- ESA WorldCover or MODIS land-cover products
- SoilGrids soil carbon layers
- Global Mangrove Watch
- FAOSTAT or national price data

## Hydroclimatic Drivers

- ERA5 precipitation and temperature
- CHIRPS precipitation
- GloFAS river discharge
- ISIMIP hydrological projections
- SPEI drought indices

## Administrative and Hydrological Boundaries

- GADM administrative boundaries
- HydroSHEDS river basins and river networks
- National boundary datasets where locally preferred

## Data-Governance Note

Raw files should be downloaded from original providers and stored locally. Large rasters, shapefiles, derived panels, and generated outputs are excluded from this public repository.
