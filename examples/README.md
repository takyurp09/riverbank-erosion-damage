# Example Extracts

These CSV files are small real-data extracts from processed outputs. They document expected columns and data shapes without releasing raw geospatial files, generated full panels, or complete result tables.

- `sample_erosion_panel.csv`: administrative-unit-year erosion and damage fields.
- `sample_damage_by_year.csv`: national damage-accounting components by year.
- `sample_river_system_summary.csv`: river-system physical erosion and damage summary.
- `sample_dsas_retreat_summary.csv`: DSAS-style retreat-rate summary by river system.

Values are intentionally limited to a few rows and should be treated as documentation examples, not as a complete replication dataset.
